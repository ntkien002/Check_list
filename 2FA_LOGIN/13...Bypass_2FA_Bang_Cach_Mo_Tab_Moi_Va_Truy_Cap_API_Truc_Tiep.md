Mô tả

Ứng dụng yêu cầu người dùng nhập mã xác thực hai yếu tố (2FA) sau khi đăng nhập. Tuy nhiên, việc kiểm tra 2FA chỉ được thực hiện ở frontend hoặc một số endpoint nhất định, trong khi nhiều API endpoint backend không kiểm tra trạng thái xác thực 2FA.

Điều này cho phép attacker:

Đăng nhập bằng username/password.

Không hoàn thành bước nhập mã 2FA.

Mở tab mới trong cùng trình duyệt.

Gửi request trực tiếp tới các API endpoint quan trọng: /Dashboard, /admin
